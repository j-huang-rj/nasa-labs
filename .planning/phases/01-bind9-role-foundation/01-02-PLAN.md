## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| bind9 templates → `/etc/named.conf` | Rendered configuration becomes privileged runtime state for a network-facing daemon. |
| inventory host mode → service behavior | `bind9_mode` decides recursion policy and query exposure. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03 | E | `templates/named.options.conf.j2` | mitigate | Render `recursion no;` on authoritative modes and a restricted resolver ACL for `allow-query` and `allow-recursion` instead of opening recursion everywhere. |
| T-01-04 | D | `tasks/service.yml` | mitigate | Run `named-checkconf /etc/named.conf` before `systemd_service` start or reload so invalid config cannot take `named` down across all three hosts. |
</threat_model>
