## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| inventory zone data -> rendered forward zone files | Structured inventory records become authoritative DNS answers served to clients |
| rendered zone files -> `named` service start | BIND loads the rendered files and either serves them or fails the service |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-04 | D | `tasks/service.yml` zone validation gate | mitigate | Run `named-checkzone` for every `bind9_zones` entry before `named` restarts so malformed forward zone files cannot take the authoritative service down. |
