## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-06 | R | `tasks/verify.yml` dig matrix | mitigate | Use source-bound dig commands (`-b 172.16.1.53` for private, `-b 127.0.0.1` for public) and assert `flags: qr aa` plus expected/no-answer conditions so verification cannot falsely pass after testing only one view. |
</threat_model>
